Estrutura Básica do HTML

Uma página HTML possui os seguintes elementos essenciais:

  • <!DOCTYPE html> - Define o documento como HTML5.
  • <html> - Contém todo o conteúdo do documento.
  • <head> - Guarda informações técnicas da página (metadados, título).
  • <title> - Define o título da aba/navegador.
  • <body> - Contém todo o conteúdo visível pelo usuário.

Exemplo Simples

<!DOCTYPE html>
<html lang=\"pt-br\">
<head>
    <meta charset=\"UTF-8\">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, mundo!</h1>
    <p>Esta é minha primeira página HTML.</p>
</body>
</html>
