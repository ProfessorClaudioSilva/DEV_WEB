Elementos Principais do HTML

Títulos (headings)

Os títulos vão de <h1> até <h6>, sendo <h1> o mais importante:

  1. Título nível h1

  2. Título nível h2

  3. Título nível h3

Parágrafos

Parágrafos são usados para escrever textos longos ou curtos. Use sempre a tag <p>.

Listas

Existem dois tipos principais de listas em HTML:

Listas ordenadas (<ol>):

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas não ordenadas (<ul>):

  • Maçã
  • Banana
  • Laranja
